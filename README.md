## 🛠️ Troubleshooting

If you experience issues, try these common fixes:

*   The interface does not show up: Ensure you run both the game and the tool as an administrator. 
*   The game crashes: Restart your computer and verify your game files in the Steam client.
*   The bot stops working: Check your internet connection. A lost signal will pause the automation tasks.
*   Missing text in tool: Download the latest Visual C++ Redistributables from the Microsoft website.
